# Gezgin Rehberi

HTML, CSS ve JavaScript ile hazırlanmış bir gezi rehberi web sitesi.

## Kullanım

`index.html` dosyasını doğrudan tarayıcıda açarak çalıştırabilirsiniz.

## Özellikler

- Tüm gezilecek yerleri listeleme
- Şehre göre filtreleme
- Detay sayfası görüntüleme
- Responsive (mobil uyumlu) tasarım
